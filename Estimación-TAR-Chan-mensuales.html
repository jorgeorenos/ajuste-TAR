Estimaci√≥n de los umbrales de inflac√≥n para la estimaci√≥n del efecto Pass through

Para la estimaci√≥n de los umbrales del modelo TAR se seguir√° la metodolog√≠a propuesta por Chang(1993). Para ello se estima el modelo para cada uno de los umbrales posibles1 teniendo como regla de decisi√≥n para escoger los umbrales verificar cuales son los modelos que minimizan la suma de residuos al cuadrado. Una vez definido el conjunto de umbrales es posible definir los distintos reg√≠menes de inflaci√≥n que determinar√°n el efecto pass through.

El modelo a estiamr es:

\[\Delta\%P_t =\beta_0 + \beta_1\Delta\%P_{t-1} + \beta_2e_t + \delta_ae_t*(\Delta\%P_{t-1}<\tau) + \beta_3\Delta\%P_{t}^E + \beta_4y_t^{brecha}\]

donde:

\(\Delta\%P_t\) Tasa de variaci√≥n del √çndice de Precios al Consumidor

\(\beta_0\) Componente aut√≥nomo de la tas de inflaci√≥n

\(\Delta\%P_{t-1}\) Componente inercial de la inflaci√≥n

\(e_t\) Tasa de variaci√≥n del tipo de cambio (tasa de depreciaci√≥n del tipo de cambio)

\(\Delta\%P_{t}^E\) Tasa de variaci√≥n del √çndice de Precios al Consumidor de Estados Unidos

\(y_t^{brecha}\) Brecha relativa del producto respecto del potencial

\(\tau\) Detona el valor del umbral para el r√©gimen de inflaci√≥n

\(\Delta\%P_{t-1}<\tau\) Variable dicot√≥mica que es 1 en r√©gimen de inflaci√≥n baja y 0 en r√©gimen de iflaci√≥n alta

Donde es claro que el uso de la dicot√≥mica separa los reg√≠menes de inflaci√≥n.

Estimaci√≥n de los distintos modelos

Como se especific√≥ la varialbe dicot√≥mica \(\Delta^{12}P_{t-1}<\tau\) ser√° 1 si el r√©gimen de inflaci√≥n es bajo o 0 si el r√©gimen de inflaci√≥n es alto, esto se replica para cad uno de los mosdelos a estimar.

# Creaci√≥n de todos los posibles umbrales
# N√∫mero de umbrales fuera por debajo o arriba de la muestra

umbrales_fuera <- length(inercial) - round(dim(datos)[1] - dim(datos)[1]*0.3, digits = 0)

# posibles umbrales del modelo

posibles <- sort(inercial)
posibles <- posibles[round((umbrales_fuera/2 + 1), 0):
                       (length(inercial)-round(umbrales_fuera/2,0))] 


# Lista de modelos

nombre_modelos <- c()
modelos <- list()
bases <- list()

# Estimaci√≥n de los modelos por cada umbral

for (i in c(1:length(posibles))) {
  
  nombre_modelos[i] <- c(sprintf("modelo_%s_%s", i, posibles[i]))
  
  m <- mutate(datos_modelo, 
                     d = ifelse(inercial<posibles[i], yes = 0, no = 1))
  
  bases[[i]] <- m
  
  modelo <- lm(bases[[i]],
                      formula = inf_gt_intermensual ~ 1 +
                        deprec*d + inercial +
                        inf_eua_intermensual + var_brecha - d)
  
  modelos[[i]] <- modelo
  
}

names(modelos) <- nombre_modelos

Luego de la estimaci√≥n de los modelos se procede al c√°lculo de la suma de residuos al cuadrado (ssr) de cada modelo.

ssr <- c()

for (i in c(1:length(posibles))) {
  
  ssr[i] <- sum(modelos[[i]]$residuals^2)
  
}

Estiamci√≥n del modelo con dos umbrales, tres reg√≠menes

Una vez determinado el conjunto de umbrales se procede a la estimaci√≥n del modelo TAR con tres reg√≠menes inflacionarios.

La nueva especificaci√≥n es:

\[\Delta\%P_t = \beta_0 + \beta_1\Delta\%P_{t-1} + \beta_2e_t + \delta_ae_t*(\Delta\%P_{t-1}<0.016929)+ \delta_be_t*(0.016929\leq\Delta\%P_{t-1}<0.05409)+ \beta_3\Delta\%P_{t}^E+ \beta_4y_t^{brecha}\]

En este caso \((0.016929\leq\Delta^{12}P_{t-1}<0.05409)\) la dicot√≥mica que es 1 si nos encontramos en el regimen de infalci√≥n media y es 0 si nos encontramos en el r√©gimen de inflaci√≥n alta, esto con el fin de establecer el reg√≠men de infalci√≥n baja acomo el escenario base.

datos_TAR <- datos_modelo %>%
